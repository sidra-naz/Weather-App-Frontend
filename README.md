Weather App
Weather App is a responsive web application built using HTML, CSS, and JavaScript. It allows users to search for any city and view real-time weather information by fetching data from a Weather API.

 Features
Search weather by city name
Display real-time temperature
Show humidity and wind speed
Display current weather conditions with icons
Responsive and user-friendly interface
Error handling for invalid city names

Technologies Used
HTML5
CSS3
JavaScript (ES6)
Weather API

Project Purpose

The purpose of this project is to practice API integration and front-end web development concepts. It demonstrates how to fetch real-time data from an external API using JavaScript and present it through a clean and responsive user interface.
